### cola

- solo se puede acceder al elemento que lleva mas tiempo ahi
- el primer objeto de la cola es el unico que tiene acceso y se puede eliminar
- solo se puede eliminar e ingresar
- se agrega al final, se elimina el primero


### Pilas
